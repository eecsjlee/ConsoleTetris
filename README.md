# ConsoleTetris

콘솔 테트리스

## 개발 환경
- Visual Studio 2022  
- .NET 8.0  
- C# 12  

## 폴더 구조
- `Program.cs`: 진입점
- `Game.cs`: 게임의 전반적인 흐름을 제어
- `Core/`: 게임 보드 및 블록 관련 클래스
- `Input/`: 키보드 입력 처리
- `Utils/`: 유틸리티 클래스
- `Constants/`: 상수 정의

## 실행 방법
1. Visual Studio 2022로 프로젝트 열기
2. `Program.cs`가 포함된 콘솔 앱으로 실행
3. `F5` 또는 `Ctrl + F5`로 실행
4. 키보드로 조작하며 플레이

## 조작 방법

| 키          | 동작            |
|-------------|-----------------|
| ← / →       | 좌우 이동        |
| ↓           | 한 칸 낙하       |
| Space       | 블록 회전        |
| P           | 일시정지         |
| R           | 게임 재시작      |
| ESC         | 종료             |
